Calculate acquisition parameters for refraction surveys

This program will calculate what the expected crossover offset is for a given set of velocities and the expected depth to the interface. This is described in the image below. The three input parameters are velocity of the overburden, velocity of the bedrock, and depth to the interface. The outputs are crossover distance, maximum geophone spacing, and maximum shot spacing.
